# Test messager app 

## Main stack
- React
- Typescript
- Redux with RTK Query (state manager)
- Antd (UI lib)
- json-server (provide mocked API)

## How to start project localy
1. Install dependencies:
```bash
yarn
```
2. Run server:
```bash
yarn api
```
3. Run front in a new terminal tab:
```bash
yarn start
```
